Writing Modes and Logical Properties

Writing Modes and Logical Properties

For the default horizontal writing mode layout, the block dimension resides along the vertical, while the inline dimension represents the horizontal. Changing the
writing-mode also changes the orientation of the inline and block dimensions.

Logical Properties are a way of referencing and applying styles to properties in the above mentioned directions.
